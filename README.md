# PhD in Mathematics Finder

A comprehensive web application to find PhD programs in Mathematics across 280+ universities worldwide.

## 🌟 Features

- **280+ Universities** across 29 regions
- **25 European Countries** (France, Italy, Spain, Germany, UK, and more)
- **Direct Application Portal** - 86 universities with no professor pre-approval needed
- **Search & Filter** functionality
- **Responsive Design** - works on all devices
- **Direct Links** to mathematics department pages

## 📁 Files to Upload

Upload these files to your GitHub repository:

1. **index.html** - Main application page
2. **styles.css** - All styling
3. **script.js** - Navigation and search logic
4. **data.js** - University database (280+ universities)
5. **direct-application-data.js** - Direct application universities
6. **direct-applications.html** - Direct application portal page
7. **README.md** - This file

## 🚀 Quick Start

### View Locally
Simply open `index.html` in your web browser.

### Deploy to GitHub Pages
1. Upload all files to your repository
2. Go to Settings → Pages
3. Select "main" branch
4. Your app will be live at: `https://yourusername.github.io/repository-name/`

### Deploy to Vercel
1. Push to GitHub
2. Import project in Vercel
3. Deploy automatically

## 🎯 How to Use

1. Open the application
2. Select a region (Canada, Australia, New Zealand, or Europe)
3. For Europe: Select a country
4. Browse universities
5. Click any university to visit their mathematics department website

## 🔍 Features

- **Search**: Filter universities by name or location
- **Hierarchical Navigation**: Easy browsing through regions and countries
- **Direct Links**: One-click access to math departments
- **Responsive**: Works on desktop, tablet, and mobile
